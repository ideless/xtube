# Development

- Install dependencies via `uv sync`
- In one terminal, run `yarn dev`
- In another terminal, run `source .venv/bin/activate` and then `yarn dev-server`

Build Linux binary:

```sh
yarn build-linux
```

Run python dir

```sh
cd build/dist/xtube
./xtube
```
